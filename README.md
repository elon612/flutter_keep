## 实现内容

* 完整的电商购物流程
* 使用 bloc 做状态管理
* 分类页面左右动画联动
* 城市选择三级联动


具体可以下载体验：

Web体验地址：https://elon612.github.io/flutter_keep/

Android 和 iOS 可以自行下载代码运行


## 项目运行环境

    1. Flutter version 2.0.6
     
    2. Dart version 2.12.3


## 注意事项

- 本项目使用 [quicktype](https://app.quicktype.io/) 网站来生成Bean。

- Web受制于js等资源过大和部署在Github上，访问会慢一些。
